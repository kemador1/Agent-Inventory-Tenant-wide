# Diccionario de campos

| Campo | Tipo | Uso actual |
| --- | --- | --- |
| `kyn_agentid` | texto | identificador funcional del agente |
| `kyn_environmentid` | texto | entorno funcional del agente |
| `kyn_displayname` | texto | nombre visible del agente |
| `kyn_createdby` | texto | nombre técnico del creador |
| `kyn_creatoremail` | texto | email resuelto del creador |
| `kyn_creatorobjectid` | texto | objeto origen del creador |
| `kyn_ownerid` | texto | nombre técnico del owner |
| `kyn_owneremail` | texto | email resuelto del owner |
| `kyn_ownerobjectid` | texto | objeto origen del owner |
| `kyn_createdin` | texto | origen de creación |
| `kyn_originnormalized` | texto | origen normalizado |
| `kyn_modelnormalized` | texto | modelo normalizado |
| `kyn_modelprovider` | texto | proveedor normalizado del modelo |
| `kyn_activityindicator` | texto | actividad operativa derivada |
| `kyn_usageindicator` | texto | indicador de uso derivado |
| `kyn_lastsyncstatus` | texto | estado de sincronización y visibilidad en origen |
| `kyn_cleanupdecision` | texto | decisión operativa de cleanup o inactivación |
| `kyn_lastseenon` | fecha-hora | última vez visto en origen |
| `kyn_missingcounter` | entero | ausencias consecutivas previas a inactivación |
